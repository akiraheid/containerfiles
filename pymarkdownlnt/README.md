PyMarkdown is primarily a Markdown linter.

Check out their [website](https://github.com/jackdewinter/pymarkdown). Consider
[donating time](https://github.com/jackdewinter/pymarkdown) to support the project.

# Usage

Recommended use.

```bash
pymarkdownlnt scan ./*.md
```

Details are at [pylint](https://github.com/akiraheid/containerfiles/blob/master/pymarkdownlnt/pymarkdownlnt).
